loadInputGTMap(gt_map_txt, splitter=':')[source]#

Load an input of GT (Ground-truth) map text and set map_list. A map text must have consistent format with the same splitter in every line like. The default format is video_file_name:gt_file_name; for example, the first line in the map file looks like video.mp4:gt.txt. Most suitable splitter is colon : because it can‚Äôt be used in a file name. Other symbol/char like = should be fine as long as it is not used in the file names of both video and GT text file.

Parameters:

  • gt_map_txt (str) ‚Äì A file path of GT (Ground-truth) map text.

  • splitter (str, default=':') ‚Äì A string with length of 1 used as a splitter of the mapping.
